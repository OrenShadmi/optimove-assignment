Clone the repository from GitHub

Open poll-service & survey service with java supported IDE

# Run poll-service
Navigate to src -> main -> optimovehomeassignment.pollsrvice 
Right click on PollServiceApplication -> Run PollServiceApplication.main() inside poll-service

# Run survey-service
Navigate to src -> main -> optimovehomeassignment.surveyservice
Right click on SurveyServiceApplication -> Run SurveyServiceApplication.main() inside survey-service

# Run optimove-client
Open optimove-client with supported client editor - Visual studio code
On the terminal run the commands:
$ npm install 
$ ng serve -o
